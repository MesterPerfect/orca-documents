Live Region Commands

The following commands can be used to access dynamically-updated content on web pages which have been made accessible through ARIA.

  • Monitor live regions: Orca Modifier+Backslash

  • Advance live region politeness setting: (Unbound)

  • Set default politeness level to off: (Unbound)

  • Go to next/previous live region in the document's reading order: D/Shift+D

  • Review the next/previous saved live region message: (Unbound)

  • Go to last live region which made an announcement: Y
